Tab 1 — Pattern & Signal

Enable as many base patterns as you like — each one runs completely independently with its own filter toggles below it. When more than one enabled pattern fires on the same candle, priority order (ATR Reversal → Engulfing → Hammer/SS → Exhaustion) picks the winner for that candle.

Early Entry: checks the still-forming candle's live (not-yet-closed) body against the pattern in the last few seconds before it closes. If it already matches, Tab 3 can enter on it immediately instead of waiting for the close. Trades already entered off an early signal are never cancelled if the true close later disagrees — only a log line records the mismatch.

Tab 3 — Trading Engine

Every trade's stake is sized live by Tab 6's Money Management settings, not a flat amount here — see the Money Management tab. A candidate whose window has already opened for more than {{ entry_deadline_sec }}s with no entry yet is skipped instead of entered late — see Tab 5's "Skipped (Late)" rows.

Danger Zone

Permanently deletes every candidate, trade, and snapshot (Tab 3/4/5/6 all go back to a clean slate — Money Management's balance and loss basket reset to your starting balance since they're computed from this same history). Currently open trades are abandoned, not settled. This cannot be undone.
